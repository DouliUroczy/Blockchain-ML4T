## 📘 About Blockchain-ML4T

Blockchain-ML4T shows how machine learning can support trading in cryptocurrencies. It follows a step-by-step process inspired by Stefan Jansen's "Machine Learning for Algorithmic Trading" method. This project tracks experiments with trading signals based on technical indicators like the MACD and RSI. It combines blockchain data with AI models to predict market trends and make decisions.

Although the technical details involve machine learning and blockchain technology, this app is built so anyone can try it out. No programming skills are needed. You just download and run the software to see how the experiment works in practice.

Key topics covered:
- Artificial intelligence applied to trading  
- Cryptocurrency data analysis  
- Popular trading indicators (MACD, RSI)  
- Use of XGBoost classifier for predictions

## 🚦 How to Use the Software

This section guides you through the basic workflow inside Blockchain-ML4T.

### Loading Data

The app automatically connects to cryptocurrency price data. It supports major coins such as Bitcoin, Ethereum, and others. You can select the coin and time frames like daily or hourly data.

### Viewing Indicators

You will see visual charts displaying the MACD and RSI indicators. These indicators help identify whether the market is trending or oversold. The app calculates these values for your selected asset period.

### Machine Learning Prediction

Behind the scenes, Blockchain-ML4T uses an XGBoost classifier. This AI model analyzes recent data patterns to predict potential market moves. The output displays simple signals like "Buy", "Hold", or "Sell".

### Interpreting Results

Signals are meant to show general trends, not exact predictions. Use them as part of your research. The interface explains each signal with brief notes on why the AI gave that advice.
